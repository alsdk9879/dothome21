번호 종류 jQuery Javascript 설명
1 태그 선택자 $("li") querySelectorAll("li") 기본 태그를 선택합니다.
클래스 선택자 $(".class") querySelector(".class") 클래스를 선택합니다.

계층 선택자
번호 종류 jQuery Javascript 설명
1 하위 선택자 $("div p") querySelectorAll("div p") div 태그 모든 자식 중 p 태그를 모두 선택합니다.
자식 선택자 $("div > p") - div 태그 모든 자식 중 첫번째 자식 p 태그만 선택합니다.
형제 선택자 $("div + p") - div 태그의 인접한 형제 p 태그 중 첫번째 태그만 선택합니다.
형제들 선택자 $("div ~ p") - div 태그의 인접한 형제 p 태그 중 모든 형제 태그를 선택합니다.

01. 기본 선택자

  • $("li")
  • $(".class")
  • $("#id")
  • $(".class, #id")
  • $("*")
리셋 클릭하면 클래스(.class)를 선택합니다. 클릭하면 아이디(#id)를 선택합니다. 클릭하면 기본(li) 태그를 선택합니다. 클릭하면 여러가지(#id, .class) 태그를 선택합니다. 클릭하면 전체 태그(*)를 선택합니다. 
{
    $(".reset1").click(function(){
        $(".view1 li").removeClass("red");
    });
    $(".btn1-1").click(function(){
        $(".view1 .class").addClass("red");
    });
    $(".btn1-2").click(function(){
        $(".view1 #id").addClass("red");
    });
    $(".btn1-3").click(function(){
        $(".view1 li").addClass("red");
    });
    $(".btn1-4").click(function(){
        $(".view1 .class, .view #id").addClass("red");
    });
    $(".btn1-5").click(function(){
        $(".view1 *").addClass("red");
    });
}

        const liList = document.querySelectorAll(".jview1 li")

            liList[i].classList.remove("blue");
        }
    });

    jbtn1_1.addEventListener("click", function(){

        //document.querySelector("#j1_id").classList.add("blue");

        //document.querySelector(".jview1 li").classList.add("blue");

        const liList = document.querySelectorAll(".jview1 li");

        //for문
        for( let i=0; i<liList.length; i++ ){
            liList[i].classList.add("blue")
        }

        //forEach문
        // liList.forEach(function(el){
        //     el.classList.add("blue");
        // });

        //for of

        //     i.classList.add("blue");
        // };

        //for in
        // for(let i in liList){
        //     liList[i].classList.add("blue");
        // };
    });
}

02. 계층 선택자

div p p p p p p p p p p p p
리셋 클릭하면 $("div p") 선택합니다. 클릭하면 $("div > p") 선택합니다. 클릭하면 $("div + p") 선택합니다. 클릭하면 $("div ~ p") 선택합니다. 
{
    $(".reset2").click(function(){
        $(".s2 span").removeClass("red")
    });
    $(".btn2-1").click(function(){
        $(".s2 span").addClass("red")
    });
    $(".btn2-2").click(function(){
        $(".s2 > span").addClass("red")
    });
    $(".btn2-3").click(function(){
        $(".s2 .select + span").addClass("red")
    });
    $(".btn2-4").click(function(){
        $(".s2 .select ~ span").addClass("red")
    });
}
